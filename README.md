## To-Do-List Application
### Open-Source Small project

### Tech-Stack:
- HTML
- CSS
- JAVASCRIPT
- NODE JS
- EXPRESS.js

- Basic back-end (like Node.js with a local storage/file-based DB).

### Future Idea
- Add drag-and-drop reordering.

### To Clone this repository:
```bash
git clone https://github.com/Jundel-Malazarte/todo-list-web-app.git
```
Use cd to Change Directory
```bash
cd todo-list-web-app 
npm install
```
Or
### Initialize your project using
Initialize your project: You need to create the package.json file.
Run the following command while in your todo-list-web-app directory:
```bash
npm init -y
```

### to run this todo list web app
make a .env file on ./todo-list-web-app/.env
and add:
```bash
PORT=3000
```
### Install Depedencies:
Install Express.js
```bash
npm install express
```

### To Run this web app:
```bash
npm start
```
